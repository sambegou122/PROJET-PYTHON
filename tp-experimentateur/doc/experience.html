experience module¶

class experience.Experience(p, m)[source]¶
__init__(p, m)[source]¶

Creates the results of an experience on p positive markers among m markers.

Parameters

  • p (int) – The number of positive markers (must be less or equal than the numbers of markers in m)

  • m (int) – The number of markers (must be greater than 0)

get_markers()[source]¶
Returns

The markers that have been tested during the experience.

Return type

Array of Marker

>>> e = Experience(6,10)
>>> m = e.get_markers()
>>> len(m)
10
>>> sorted([ repr(i) for i in m])
['m0', 'm1', 'm2', 'm3', 'm4', 'm5', 'm6', 'm7', 'm8', 'm9']
get_positive_markers()[source]¶
Returns

The positive markers.

Return type

Array of Marker

>>> e = Experience(10,100)
>>> p = e.get_positive_markers()
>>> len(p)
10
